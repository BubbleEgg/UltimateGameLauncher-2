# UltimateGameLauncher-2
A launcher that manages all your games and accounts from other platforms like Steam, Battle.net, Origin...

<img align="center" width="80%" src="https://ugl.seemslegit.me/screenshots/1.png?t=186485461641">

## Current Features

**NOTE:** This program is very early in development, so feel free to give feedback.

### Games
- List games from: Steam, Battle.net
- Start the games with the selected user
- Desktop shortcuts for starting games

### Users
- Manage users from: Steam, Battle.net
- Steam:
    - Switch users
    - Generate Steam Guard codes
    - List of Steam confirmations/trades
    - Accept or deny confirmations
    - User data is stored encrypted
    - Only requires username and Steam64ID to switch users
- Battle.net:
    - Switch users

### Other
- Update system

### Coming Soon
- Translations
- Manage games and users from other platforms
